数据导出与备份的区别
    数据导出，导出的纯粹是业务数据
    数据备份，备份的是数据文件、日志文件、索引文件等等
        全量备份 增量备份

数据导出的分类
    SQL 文档：
        mysqldump 用来把业务数据导出城 SQL 文件，其中也包括了表结构
        mysqldump -uroot -p [--no-data] 数据库 > 路径
    文本文档：
         只有单纯的数据，所以速度更快，数据量大的时候推荐使用（图形界面里面"导出向导"，选 text）记得要先导出一份数据表结构文件，避免数据表结构丢失。

导入 SQL 文件
    source 命令用于导入 SQL 文件，包括创建数据表，写入记录等
    USE demo;
    SOURCE backup.sql;
